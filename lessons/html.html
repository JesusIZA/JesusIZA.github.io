Урок 1. Вступ

Завантажити

HTML (Hyper Text Markup Language) - стандартна мова розмітки веб-сторінок в Інтернеті. Більшість веб-сторінок створюються за допомогою мови HTML. Документ HTML оброблюється браузером та відтворюється на екрані у звичному для людини вигляді

Для того, щоб створити html-документ, достатньо створити звичайний текстовий файл (документ.txt) і змінити його розширення на html (документ.html)

Html-документ складається з тегів (<відкриваючий тег></закриваючий тег>). Корененевий тег має назву html (<html></html>). У кореневому є два підтеги head (<head></head>) та body (<body></body>)

Тег head - містить інформацію необхідну браузеру для правильного відображення контетну сторнки (мову, стилі, основні скрипти та інше)

Тег body - містить контент сторінки (текст, зображення, таблиці тощо)

Коментарі (<!--Коментар до коду-->) - спеціальні теги, які браузер не відображає на сторінці, вони слугують, для пояснення коду тому, хто буде його читати (самому розробнику, розробнику, який в подальшому підтримуватиме сайт) 

<html>
  <head>
    <meta charset="utf-8">
  </head>
  <body>
    <!--Тестовий код-->
    Контент сайту
  </body>
</html>
<meta charset="utf-8"> - встарновлення кодування сторінки
