    Курси програмування з репетитором

    WEB-frontend

    • HTML5 - повне вивчення технології; входять структурні теги, теги для тексту, форми та багато іншого. Після вивчення цієї технології Ви навчитеся правильно будувати структуру сторінки.

    • CSS3 - вивчення основ створення каскадних таблиць стилів, а також додаткового модуля flex. Вивчивши цю технологію та застосувавши її до html-сторінки, Ви вдихнете у неї життя. Ваш сайт буде не просто красивим, він буде адаптуватися під різні пристрої (планшети, телефони тощо).

    • JavaScript - під час проходження цієї мови програмування, Ви отримаєте не лише основи мови, але і грунтовні знання роботи з DOM-деревом документу, створення та застосування регулярних виразів і безліч додаткових тем. Пройшовши курс по JavaScript Ви зможете динамічно змінювати структуру сторінки. Використавши цю технологію у розробці сайту Ви зробите його не просто живим, але й інтерактивним.

    • jQuery - курс по найпопулярнішій бібліотеці JavaScript включає вивчення методів роботи з DOM-деревом, стилями (CSS), анімацією, подіями, а також форматом JSON та AJAX запитами. Знання цієї технології допоможуть Вам писати JavaScript код ефективніше та лаконічніше. Бібліотека заставить Вас буквально полюбити JavaScript.

    • Wordpress - пропонуємо ознайомлення з найпопулярнішою системою керування вмісто. З цією технологією Ви зможете створити адаптивний сайт з сучасним дизайном, не написавши ні рядка коду.

    Після проходження курсу Ви зможете створювати правильно побудовані, красиві, адаптивні та інтерактивні сайти.

    PHP & Database

    • PHP - вивчення основ мови програмування, включно з темою ООП (об'єктно орієнтоване програмування). Після проходження цієї частини Ви зможете створити повноцінну серверну частину сайту, обробляти POST і GET запити, працювати з сесією тощо.

    • phpMyAdmin - освоєння знань для роботи з базами даних (створення, зміна та інше).

    • SQL - ознайомлення з мовою структурованих запитів. Вивчення базових методів роботи з таблицями (створення, зміна тощо).

    Освоївши курс Ви зможете створити повністю функціональниу серверну частину сайту, зберігати інформацію в базі даних та працювати з нею.

    Java full

    • Java full - повний курс вивчення мови програмування. Передбачає грунтовне вивчення основ мови, вичення та застосування основних принципів ООП (абстракція, поліморфізм, інкапсуляція). У курс входять додаткові модулі, такі як: Java Servlet - базові знання розробки сайтів, JDBC - знання по роботі з базами даних та бонусний модуль Алгоритми - навчання розв'язувати різноманітні алгоритмічні завдання (пошук у масиві, сортування та багато іншого), модуль спрямований не просто научити програмувати, а привчити алгоритмічно мислити.

    • JSP - освоєння знань для динамічної роботи з веб-сторінками. Ознайомлення з Tomcat сервером та системою збору проектів Maven

    • PostgreSQL/H2 - освоєння знань для роботи з базами даних (створення, зміна та інше).

    • SQL - ознайомлення з мовою структурованих запитів. Вивчення базових методів роботи з таблицями (створення, зміна тощо).

    • JavaFX - вивчення сучасного способу створення кросплатформених додатків зі зручним дизайном.

    Курс Java full забезпечить Вас міцною основою для вивчення та розробки кросплатформенних додатків. Після закінчення курсу Ви не тільки вивчитемову основи мови програмування, Ви навчитеся мислити як програміст, мислити алгоритмічно, що є набагато важливішим за знання синтаксису.

    Python

    • Python - вивчення базових знань мови. У курс входять теми: робота зі списками, рядками, функціями; іпморт різноманітних модулів; обробка пимилок та безліч інших.

    Пройшовши курс Ви отримаєте базові знання мови програмування за відносно короткий час.

    Node.js

    • Node.js - основи технології: події, обробка запитів, потоки, робота з файлами, робота зі статичними файлами тощо. Курс передбачає ознайомлення з наступними модулями: Express, EJS шаблонізатор.

    Ви ознайомитесь з основами мови програмування, вивчите кілька додаткових модулів, отримаєте достатні знання для подальшого самостійного поглибленого вивчення.

    Scratch

    • Scratch - навчання дітей програмуванню та алгоритмічному мисленю у процесі створення веселих ігор. Що може бути цікавішим для дитини ніж гратися у ігри? Створювати Ігри!

    Курс підходить для дітей будь-якого віку. Програмування може бути цікавим навіть для дітей!

    Blender3D

    • Blender3D - вивчення 3D моделювання за допомогою програми Blender3D. Крім стандартних знань інтерфейсу програми Ви освоїте роботу з освітленням, матеріалами та текстурами, застосування і налаштування різноманітних модифікаторів (Скручування, Дзеркальне відображення тощо).

    Ви отримаєте хороші знання не тільки роботи у програмі Blender3D, а і 3D моделювання в цілому. Навчитеся імпортувати файли різних форматів та експортувати об'єкти у інші програми.

    UnrealEngine

    • UnrealEngine - навчання створенню 3D ігор за допомогою одного з найновіших ігрових двигунів UnrealEngine. Пройшовши курс Ви отримаєте базові знання по створення різного роду ігор (від третього лиця, від першого лиця, платформери та інші), освоїте роботу по створенню ландшафту, моделюванню 3D об'єктів, научитеся створювати та працювати зі змінними, функціями, BluePrint технологією загалом та багато іншого.

    Курс дасть Вам базові знання роботи у 3D ігрових двигунах, 3D моделювані, познайомить з методами обробки освітлення, створення та застосування змінних, функцій, обробки різноманітних подій тощо.

    Розробка сайтів на замовлення

    Односторінкові інформаційні сайти

    • Односторінковий сайт - сайт, який складається з однієї строніки, зазвичай це сайт-візитка. Такий сайт виглядає лаконічно, інформація розбита на блоки, для переміщення до блоку з певною інформацією використовується меню. Сайт повністю адаптивний (кореткно відображається на комп'ютерах, планшетах та телефонах). За бажанням можна добавити слайдери чи просто анімацію, форму зворотньго зв'язку, кнопку зателефонувати мені, відео, фото, карти тощо.

    Односторінковий сайт це зручне, лаконічне та відносно дешеве рішення для розміщення невеликої кількості інформації.

    Багатосторінкові інформаційні сайти

    • Багаторінковий сайт - складається з декількох сторінок, потрібен у випадках коли односторінкового недостатньо для розміщення усієї інформації. Такий сайт може містити усе, що і односторінковий (відео, фото, слайдри, кнопку зателефонувати мені, карти, форми зворотнього зв'язку тощо). Сайт повністю адаптивний (кореткно відображається на комп'ютерах, планшетах та телефонах).

    Багатосторінковий сайт - рішення для сртуктурованого та зручного розміщення великої кількості інформації. Такі сайти можуть мати наступну структуру: головна, послуги, про нас, партнери, контакти тощо.
