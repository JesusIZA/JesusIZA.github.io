Розробка сайтів на замовлення

Односторінковий сайт - сайт, який складається з однієї сторінки, зазвичай це сайт-візитка. Такий сайт виглядає лаконічно, інформація розбита на блоки. Для переміщення до блоку з певною інформацією використовується меню.

Зручне, лаконічне та відносно дешеве рішення для розміщення невеликої кількості інформації.

Багатосторінковий сайт - складається з декількох сторінок, потрібен у випадках, коли односторінкового недостатньо для розміщення усієї інформації.

Рішення для структурованого та зручного розміщення великої кількості інформації.

Курс Java з репетитором

Java основи - повний курс вивчення мови програмування. Передбачає грунтовне вивчення основ мови та застосування основних принципів ООП (абстракція, поліморфізм, інкапсуляція). У курс входять додаткові модулі: Java Servlet, JDBC та бонусний модуль Алгоритми.

Курс забезпечить Вас міцною основою та навчить мислити алгоритмічно.

Java додатково - курс містить: JSP - динамічна робота з веб-сторінками, PostgreSQL/H2 - робота з базами даних, SQL - методи роботи з таблицями, JavaFX - сучасний спосіб створення кросплатформених додатків зі зручним дизайном.

Після закінчення курсу Ви зможете розробляти кросплатформенні додатки.

Курс PHP та Бази Даних з репетитором

PHP основи - вивчення основ мови програмування, включно з темою ООП (об'єктно-орієнтоване програмування), обробка POST і GET запитів, робота з сесією тощо.

Ви зможете створити повноцінну серверну частину сайту.

SQL та phpMyAdmin - освоєння знань для роботи з базами даних. Вивчення базових методів роботи з таблицями (створення, зміна тощо).

Зберігання інформації у базі даних та робота з нею.

Курс WEB front-End з репетитором

HTML5 та CSS3 - повне вивчення технології HTML (структурні теги, теги для тексту, форми та інші). Освоєння основ створення каскадних таблиць стилів, а також додаткового модуля flex.

Ви навчитеся правильно створювати красиві, анімовані та адаптивні сайти.

JavaScript та jQuery - базові знання мови програмування, методи роботи з DOM-деревом документу, створення та застосування регулярних виразів, робота зі стилями (CSS), анімацією, подіями, форматом JSON та AJAX запитами тощо.

Ефективне створення інтерактивних сайтів.

Wordpress - ознайомлення з найпопулярнішою системою керування вмістом. З цією технологією Ви зможете створити адаптивний сайт з сучасним дизайном, не написавши ні рядка коду.

Створення сучасного адаптивного сайту не програмуючи.

Курс 3D-моделювання та розробки ігор з репетитором

Blender3D - крім стандартних знань інтерфейсу програми Ви освоїте роботу з освітленням, матеріалами та текстурами, застосування і налаштування різноманітних модифікаторів тощо.

Ви отримаєте базові знання 3D-моделювання.

UnrealEngine - базові знання по створення різного роду ігор (від третього лиця, від першого лиця, платформери та інші), освоїте роботу по створенню ландшафту, моделюванню 3D-об'єктів, научитеся створювати та працювати зі змінними, функціями, BluePrint технологією загалом та багато іншого.

Створення 3D-ігор за допомогою одного з найновіших ігрових двигунів UnrealEngine.

Інші курси програмування з репетитором

Python - вивчення базових знань мови. У курс входять теми: робота зі списками, рядками, функціями; імпорт різноманітних модулів; обробка помилок та іншие.

Ви отримаєте базові знання мови програмування за відносно короткий час.

Node.js - основи технології: події, обробка запитів, потоки, робота з файлами, робота зі статичними файлами тощо. Курс передбачає ознайомлення з наступними модулями: Express, EJS-шаблонізатор.

Достатні знання для подальшого самостійного поглибленого вивчення.

Scratch - навчання дітей програмуванню та алгоритмічному мисленню у процесі створення веселих ігор. Що може бути цікавішим для дитини ніж грати у ігри? Створювати їх!

Курс підходить для дітей будь-якого віку.
